# LeetCode Solutions

This repository contains solutions to LeetCode and NeetCode problems.

## Structure

- `neetcode-150`: Solutions to NeetCode 150 problems
- `leetcode`: Solutions to LeetCode problems

## Installation for LeetDocs CLI 
(automating documentation generation)

```bash
cd leetdocs-cli
npm install
npm link
```

## Usage

```bash
leetdocs new -n
```
