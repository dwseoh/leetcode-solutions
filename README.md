# LeetCode Solutions

This repository contains solutions to LeetCode and NeetCode problems.

## Structure

- `neetcode-150`: Solutions to NeetCode 150 problems
- `leetcode`: Solutions to LeetCode problems

## Installation for LeetDocs CLI 
(automating documentation generation)

```bash
cd leetdocs-cli
npm install
npm link
```

## Usage

```bash
leetdocs new -n
```

after completing the problem, run:
```bash
leetdocs done [problem_id] -n
```

to run complexity analysis + autofill some parts of the metadata + README for the problem
